⚙️ 상품 카탈로그 + 단가

원가 = 본인이 공급사에 지불하는 매입가. 기본 납품가 = 일반 입력자한테 받는 가격. 마진 = (납품가 − 원가) × 수량.

📦 입고 기록 (배치별 단위원가)

매입가/물류비가 매번 달라도 OK. 매입가는 지난번 값이 자동 입력되고, 바뀌었으면 덮어쓰면 됩니다. 물류비는 비워두면 0으로 계산. 등록하면 이후 새 주문에 새 원가가 자동 적용됩니다 (과거 주문은 보존).

상품 선택 → 수량·매입가 입력 → 단위원가 자동 계산

🔑 사용자 비밀번호 재설정 / 푸시 알림

입력자가 비밀번호 잊었을 때 — Supabase 대시보드에서 직접 재설정합니다.

  1. 아래 [Supabase 사용자 관리] 버튼 → Authentication → Users
  2. 해당 사용자 이메일 행 우측의 ··· 점 3개 → Reset password
  3. 새 비밀번호 입력 (6자 이상)
  4. 입력자한테 새 비번 알려주기 (카톡 등)

🔑 로그인 이메일·비밀번호 변경

Supabase Auth에서 처리합니다 (보안상 앱 내부에서는 불가).

  1. 아래 버튼 → Authentication → Users → 본인 행 점 3개
  2. Change email → 새 이메일 입력 → 인증 메일 확인 → 클릭
  3. 또는 Reset password → 새 비밀번호 입력
  4. 다음 로그인부터 새 정보 사용
↗ Supabase 사용자 관리 열기

👑 관리자 권한 다른 사람한테 넘기기

새 사람을 일반 회원가입 → 위에서 권한 변경.

  1. 새 관리자가 앱에서 회원가입 (이름·비밀번호) → 「승인 대기」 떠야 함
  2. Supabase SQL Editor에서:
    update profiles set role='admin' where display_name='새관리자이름';
    update profiles set role='input' where display_name='기존관리자이름';
  3. 새 관리자가 재로그인하면 관리자 화면 뜸
